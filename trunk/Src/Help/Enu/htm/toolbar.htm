Product Contact

1. Standard Toolbar
 
    New Strategy ------------ Create a new strategy;
    Open Strategy ----------- Open an existing strategy;
    Workspace --------------- Show or hide the strategy workspace;
    Settings ---------------- Common settings;
    View -------------------- Show views;
    FullScrn ---------------- Toggles Full Screen on and off;
    Indicator --------------- Show indicators;
    Period ------------------ Show graph of different period;
    Download ---------------- Auto download historical data;
    Data -------------------- Data management, installing packages, creating packages, managing Dividend and Ration data and financial data;
    Export ------------------ Export historical data to Access Database, Excel table or text file;
    Help -------------------- Show help;
 
2. View Toolbar
 
    Page Left --------------- Scroll the historical graph page left;
    Page Right -------------- Scroll the historical graph page right;
    Move Left --------------- Move focus left;
    Move Right -------------- Move focus right;
    Zoom in ----------------- Zoom in the historical graph;
    Zoom out ---------------- Zoom out the historical graph;
    Prev Stock -------------- Show previous stock;
    Next Stock -------------- Show next stock;
    Add Forecast ------------ Add forecast data;
 
3. Period Toolbar
 
    Day --------------------- Show day-peirod historical graph;
    Week -------------------- Show week-peirod historical graph;
    Month ------------------- Show month-peirod historical graph;
    5 Minutes -------------- Show 5-minute-period historical graph;
    15 Minutes ------------- Show 15-minute-period historical graph;
    30 Minutes ------------- Show 30-minute-period historical graph;
    60 Minutes ------------- Show 60-minute-period historical graph;
 
4. Strategy Toolbar
 
    Active Strategy Property -- Setup the active strategy's properties;
    Modify Active Strategy Stocks -- Modify the active strategy stocks;
    Active Strategy Rate Settings -- Setup the active strategy rate;
    Active Strategy Indicator Rule Settings -- Setup the active strategy indicator rule;
 
    Show Active Strategy ---- Show the active strategy;
    Show Active Strategy Report -- Show the active strategy report;
    Show Active Strategy Trade -- Show the active strategy trade;
 
5. Stock Toolbar
 
    Index List -------------- Show list of all the indexes;
    Strategy Stocks --------- Show list of the active strategy stocks;
    Group List -------------- Show group's stock list;
 
    Domain List ------------- Show domain's stock list;
 
    Class List -------------- Show classic stock list;
    List Columns Settings --- Setup stock list columns;
