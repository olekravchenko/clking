3. Stock list Accelerator
 
    Ctrl+A ---------- Select All;
    Ctrl+C ---------- Copy;
    Ctrl+P ---------- Print.
 
4. Intradays Graph Accelerator
 
    Ctrl+P ---------- Print;
    Home ------------ Move home;
    End ------------- Move end;
    PageUp ---------- Show previous stock in intradays price line graph; Scroll page up in bargain detail graph;
    PageDown -------- Show next stock in intradays price line graph; Scroll page down in bargain detail graph;
    ESC ------------- Cancel selection;
    Left ------------ Move focus left;
    Right ----------- Move focus right;
    Up -------------- The report region page up;
 
    Down ------------ The report region page down;
 
    Double click one of the Big-Trade can show the corresponding stock.
 
5. Historical Graph Accelerator
 
    Ctrl+P ---------- Print;
    Ctrl+Left ------- Page left;
    Ctrl+Right ------ Page right;
    Home ------------ Move home;
    End ------------- Move end;
    PageUp ---------- Previous stock;
    PageDown -------- Next stock;
    F8 -------------- Toggle period;
    ESC ------------- Cancel Selection;
    Left ------------ Move left;
    Right ----------- Move right;
    Shift+Left ----- Move left and select also;
 
    Shift+Right ---- Move right and select also;
 
    Shift, Ctrl ---- Make multi-selection;
 
    Enter ----------- Edit the predictive day;
 
    Up -------------- Zoom in; Increase the price of predictive day;
 
    Down ------------ Zoom out; Decrease the price of predictive day;
 
    Shift+Up ------- Increase the volume of predictive day;
 
    Shift+Down ----- Decrease the volume of predictive day;
 
    Select a region and zoom in it.
 
6. Base Information Accelerator
 
    Ctrl+A ---------- Select All;

    Ctrl+P ---------- Print;
    Ctrl+F ---------- Find.
